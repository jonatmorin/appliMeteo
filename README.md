```
npm run build
```

```
npm run lint
```

### API key
```
To use this app, get an API key from https://openweathermap.org/
Inside root folder, create a file named ".env.local" with this line of code: 

VUE_APP_WEATHER_API_KEY =  [apikeywithnobrackets]
```
